## 📂 Homework Assignments & Pages

*   **HW01: SET50 Major Shareholders Network Dashboard**
    *   **Live Page:** [https://summer-sna.streamlit.app/HW01_SET50](https://summer-sna.streamlit.app/HW01_SET50)
    *   **Description:** An interactive dashboard that scrapes top 5 major shareholders for SET50 companies, computes network statistics and centralities (Degree, Betweenness, Eigenvector), and visualizes the co-ownership stock-stock projection.

## 🚀 Running Locally

### 1. Clone the repository
```bash
git clone https://github.com/6720422030/DADS7201_Streamlit.git
cd DADS7201_Streamlit
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Launch the application
```bash
streamlit run Home.py
```
